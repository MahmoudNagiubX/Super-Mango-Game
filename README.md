**Super Mango** is a retro-style **2D platformer game** built using **C++ and Raylib**.  
The project demonstrates key **game development and software engineering principles**, including object-oriented programming, game loops, physics simulation, collision detection, input handling, and modular architecture.

This project was designed to showcase **low-level graphics programming, structured game design, and C++ engineering practices**.

---

# 🎮 Game Overview

Super Mango is a fast-paced **2D platform adventure** where the player controls a small character navigating through dangerous environments filled with obstacles, enemies, and traps.

The objective is to:

- Collect coins
- Avoid enemies and traps
- Survive each level
- Achieve the highest possible score

The game features **pixel-art graphics**, **responsive controls**, and **multiple environments**.

# 🏗️ Project Architecture

The game follows a **modular architecture** to keep the code scalable and maintainable.
```
SuperMango/
│
├── assets/ # Textures, sprites, backgrounds
│
├── src/
│ ├── main.cpp
│ ├── Game.cpp
│ ├── Player.cpp
│ ├── Enemy.cpp
│ ├── Level.cpp
│ ├── Collision.cpp
│
├── include/
│ ├── Game.h
│ ├── Player.h
│ ├── Enemy.h
│ ├── Level.h
│
└── README.md
```

### Core Components

| Module | Responsibility |
|------|------|
| **Game** | Controls the main game loop and scenes |
| **Player** | Handles movement and physics |
| **Enemy** | Enemy behavior and interactions |
| **Level** | Environment and platform layout |
| **Collision** | Physics and collision detection |

# 🎮 Game Mechanics

## Player Movement

The player can:

- Move left and right
- Jump across platforms
- Avoid enemies and hazards

Movement uses **basic gravity simulation and jump physics**.

---

## Collision System

The game includes a custom collision system that manages:

- Platform collisions
- Enemy interactions
- Hazard detection
- Map boundaries

---

## Score System

Players earn points by:

- Collecting coins
- Surviving enemy encounters
- Progressing through the level

The **HUD displays player lives and score**.
